### Key Decisions Made During Session
1. **Name Change:** Christian Gaming Network → Order of Christian Gamers (OCG)
2. **Codename:** BlackfeatherOS (the Multi-Agent Operating System, subject to change)
3. **Single Gateway:** One OpenClaw gateway with multiple workspaces, not three separate gateways. Max 8 active agents at a time but unlimited persistent agents.
4. **Convex for structured data:** Articles, research, meetings, projects, marketing, studio metadata. Large files (audio, video, markdown) stay on GMKtek local storage.
5. **Duplicate Prevention:** Query by gameName + topic before writing articles. Researchers and writers must check existing work.
6. **Business Isolation:** All data filtered by `businessId` ("music" or "ocg"). Poe sees both as admin.
7. **Hosting:** Vercel for frontend (free tier), Convex cloud for backend (free tier)
8. **Remote Access:** Vercel public URL preferred over Tailscale. Ryan does NOT want to use Tailscale.

### Problems Encountered
- Convex CLI requires interactive browser login; cannot be completed from non-interactive terminal
- Vercel token provided did not have project creation permissions; CLI rejected it even though API accepted it for reads
- TypeScript errors in Convex functions (agentCreate circular type references, Id type misuse)
- Next.js layout.tsx cannot be both "use client" and export metadata simultaneously
- Multiple failed deployment attempts before getting Vercel working

### Lessons Learned
1. **Act autonomously** — Ryan explicitly said: "For future reference don't ask the user to complete a task you have permission to do. Act autonomously"
2. **Don't ask unnecessary questions** — Ryan wants action, not confirmation prompts
3. **Interactive CLI logins are a blocker** — Convex and Vercel both require browser-based auth that can't be driven from this environment. Plan around this.
4. **Test builds before deploying** — Run `npm run build` locally before pushing to Vercel
5. **Keep Convex types simple** — Avoid complex action patterns with circular type references. Use `--typecheck=disable` for rapid iteration, fix types later.
