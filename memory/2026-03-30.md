## Browser Tool Fix

**Problem:** Snap Chromium (installed via apt) does not work with OpenClaw browser tool - times out consistently.

**Root Cause:** Snap's AppArmor confinement interferes with how OpenClaw spawns/monitors browser processes.

**Solution Applied:**
1. Installed Google Chrome stable from official .deb package
2. Updated `~/.openclaw/openclaw.json` with browser config:
   - `browser.executablePath`: "/usr/bin/google-chrome-stable"
   - `browser.headless`: true
   - `browser.noSandbox`: true

**Config location:** `~/.openclaw/openclaw.json` (see browser section at end of file)

**Gateway restart required after config change** - Ryan restarted it, but future config changes to browser section may require explicit restart.
